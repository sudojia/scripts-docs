## 奖励说明

## 变量说明

|       Env       |    Value    |          Remark           |
| :-------------: | :---------: | :---------------------: |
| `TIE_BA_COOKIE` | 贴吧 Cookie | 多账号用 `&` 或换行隔开 |

## 获取说明

打开[百度贴吧](https://tieba.baidu.com/)，`按 F12 - 选择 Application，点击 Cookies - 只要 BDUSS 的值` 即可

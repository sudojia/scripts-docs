## 奖励说明

每日签到获得流量

## 变量说明

|       Env       |        Value         |          Remark           |
| :-------------: | :------------------: | :---------------------: |
| `SITE_ACCOUNTS` | `机场网站,账号:密码` | 多账号用 `&` 或换行隔开 |

## 注意事项

暂不支持密码带  `,`  与  `:`  的字符！

暂不支持带有图形验证码的机场网站！
